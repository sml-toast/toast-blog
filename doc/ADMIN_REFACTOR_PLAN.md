> 2026-06-18 | 状态: 待审批

## 一、变更概述

| 模块 | 当前行为 | 目标行为 |
|------|---------|---------|
| 环境切换 | 无视觉效果 | DEV=🟢绿 / TEST=🟠橙 / PROD=🔵蓝，色块标签 |
| 数据目录 | localStorage 不同 key | `data/{env}/` 目录 + 文件备份 |
| 备份 | 下载 JSON 文件 | 保存到 `data/{env}/YYYY-MM-DD_HHMMSS/` |
| 还原 | 选择 JSON 文件 | 下拉列表选备份目录 |
| 导出 | 单 JSON 文件 | ZIP 压缩包（数据+图片+附件） |
| 导入 | 单 JSON 文件 | 解压 ZIP 导入 |
| 登录缓存 | sessionStorage | localStorage + 30min 过期 |
| 前台显示 | 环境切换不可见 | 导航栏显示当前环境标签 + 语言切换 🌐 |

## 五、开发规则

1. 每完成一个功能 → 测试 → 记录 DEVELOPMENT_LOG.md
2. 测试通过才能继续下一个
3. 修改文件前备份 → 后缀 `.bak`
4. 每次提交前 `git status` 检查
5. 提交信息格式: `{type}: {模块} — {改了什么}`
